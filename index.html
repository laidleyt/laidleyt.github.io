NYC Taxi: ML Forecasting & Anomaly Dashboard

Dynamically-updating dashboard with Prophet forecasts, DBSCAN-based anomaly detection, and Bayesian counterfactuals (Orbit). Data ingested monthly from NYC Taxi and Limo Commission (TLC). Built in Python with Plotly/Dash, DuckDB, Prophet, DBSCAN, and Orbit (June 2025).

NYC Taxi Revenue & Mileage Dashboard: Historical 2011-2024 Time Series

Interactive dashboard analyzing trip revenue and mileage trends from 2011–2024 using TLC data. Includes vendor/payment breakdowns and fixed layout for interpretability. Data handling was done with DuckDB, while dashboard construction was handled with Plotly/Dash (May 2025).

US-County Level Demographic Changes Map, 2018-2023

This mapping dashboard visualizes changes in college educated residents, age 25-34, participating in the labor force, and total households at the US county-level. It was built using Plotly/Dash, and a toggle switch feature highlights the top/bottom 300 counties in each metric (May 2025).
